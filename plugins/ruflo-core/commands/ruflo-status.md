$ARGUMENTS
Run diagnostics and show system status.
```bash
npx @sparkleideas/cli@latest doctor
npx @sparkleideas/cli@latest status
```
To auto-fix issues, run `npx @sparkleideas/cli@latest doctor --fix` separately.
