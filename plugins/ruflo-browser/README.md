# ruflo-browser

Agentic browser automation with Playwright for testing, scraping, and UI interaction.

## Install

```
/plugin marketplace add ruvnet/ruflo
/plugin install ruflo-browser@ruflo
```

## Features

- **Browser testing**: Navigate, click, fill forms, and validate UI with 23 browser tools
- **Screenshots**: Capture visual state for regression testing
- **Web scraping**: Extract structured data with DOM queries and accessibility trees
- **Session management**: Multiple concurrent browser sessions

## Commands

- `/browser` -- List active sessions, take screenshots, manage tabs

## Skills

- `browser-test` -- Automated UI testing with Playwright
- `browser-scrape` -- Extract structured data from web pages
