# hive-mind

Hive Mind collective intelligence system for advanced swarm coordination.

## Usage
```bash
npx @sparkleideas/cli@latest hive-mind [subcommand] [options]
```

## Subcommands
- `init` - Initialize hive mind system
- `spawn` - Spawn hive mind swarm
- `status` - Show hive mind status
- `resume` - Resume paused session
- `stop` - Stop running session

## Examples
```bash
# Initialize hive mind
npx @sparkleideas/cli@latest hive-mind init

# Spawn swarm
npx @sparkleideas/cli@latest hive-mind spawn "Build microservices"

# Check status
npx @sparkleideas/cli@latest hive-mind status
```
