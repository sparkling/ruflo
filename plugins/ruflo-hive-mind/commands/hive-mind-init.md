# hive-mind-init

Initialize the Hive Mind collective intelligence system.

## Usage
```bash
npx @sparkleideas/cli@latest hive-mind init [options]
```

## Options
- `--force` - Force reinitialize
- `--config <file>` - Configuration file

## Examples
```bash
npx @sparkleideas/cli@latest hive-mind init
npx @sparkleideas/cli@latest hive-mind init --force
```
